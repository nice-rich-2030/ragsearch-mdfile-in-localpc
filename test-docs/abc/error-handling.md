## 💡VS CODEのショートカット

- ファイル先頭/末尾へ移動: Ctrl + Home / End
- 単語単位でカーソル移動: Ctrl + ← / →
- 選択行を上下に移動: Alt + ↑ / ↓

## VS CODEのフォントを少し小さくしたい。
VS Codeのフォントを小さくするには、設定画面（Ctrl+, / Mac: Cmd+,）を開き、「font size」と検索して「Editor: Font Size」の数値を小さく（デフォルト12〜14px）設定します。一時的な変更であれば、Windows/LinuxはCtrlと-、MacはCmdと-のキー操作で画面全体のズームアウトが可能です。
#フォント #設定 #カスタマイズ
